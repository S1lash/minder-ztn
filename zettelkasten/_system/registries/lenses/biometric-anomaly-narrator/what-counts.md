# What counts (biometric-anomaly-narrator)

- Single-day **medium+** deviation flagged in yesterday's record's
  `## Baseline Deviations` section.
- Categorical event (HRV status / training status / ACWR / readiness
  level transition) on yesterday's date.
- Streak start (concept appeared in today's `## Streak Transitions`).
- Streak end / recovery (recovery_after_* in today's `## Streak Transitions`).
- Cross-domain link to journal: same-date observation or meeting body
  carrying narrative context that explains the deviation (anxiety
  marker → low HRV; conflict marker → low sleep; late-work marker
  → low recovery).
