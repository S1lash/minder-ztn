# What counts (opportunity)

New leads; the open-doors change-layer supports it.

- **A new opportunity this week** — an event/node that emerged this week and moves a stated
  goal. Surface every one that clears the quality bar (real signal + falsifier + not
  role-routine) — no artificial count cap. Each: what it actually is (with context) → where
  it could lead (branch, plain-language mechanism, bounded downside + doors it opens) → a
  one-week cheap test move.
- **One genuine opportunity-bearing weak tie who entered orbit this week** — not already in
  the base (texture gate first), from outside the work hierarchy, carrying a mobilizable
  resource that maps to a real goal, reachable — a coherent picture, not a tally of signals.
  Consuming knowledge (heard a speaker, studied a project) is NOT a tie entering orbit.
  A cheap first move — something to build/try OR an honest probe-conversation — is part of it.
  Strongest when the person is native to a world the owner's goals actually try to bridge.
- **A material change in an open door this week** — advanced / running N weeks + what changed /
  decayed / closed / absorbed. Delta-focused. An old door that became materially relevant this
  week is promoted into the current reading as live.
- **A real fork the week sharpened** — a course-choice (not a delivery task), by reversibility.
- **The soft execution mirror** — one dry factual line, only if the SOUL documents an
  execution-gap pattern AND ≥3 open doors have decayed. Data, not reproach; never the lead.
- **"Nothing new this week"** — no new opportunity clears the bar and open doors are unchanged.
  A valid, honored output.

Every surfaced item must read cold — self-contained context, zero jargon (see prompt's Output
contract): who a named person is + how they entered, what an opportunity actually is, what the
prior state was before the delta.
